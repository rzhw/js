documentation

Using just a bit of code, you've got yourself a basic gallery. (In order: HTML, CSS, JavaScript)

<div id="demo-basic"></div>
#demo-basic .images
{
	width: 320px;
	height: 240px;
}
$(document).ready(function() {
	$("#demo-basic").glyr({
		images: [
			{ src: 'rose1.jpg', alt: 'Rosa bracteata' },
			{ src: 'rose2.jpg', alt: 'Bandel rose' }
		]
	});
});

But basic is just that, basic. For the most part, you'll be making your gallery look special through CSS. A numerous amount of CSS classes are provided for your use. These classes can have a prefix prepended in the event you have colliding class names, but that will be covered later.

  • .glyr - This contains everything part of one gallery, created from whatever element you run the glyr method on. Note: this class name is not affected if you set a prefix
  • .images - The container for the images.
  • .image - An image.
  • .bottom - The container for the tabs/caption.
  • .tabs - The container for the tabs.
  • .tab - A tab.
  • .tab-on - A tab that represents the currently shown image.
  • .tab-off - A tab that represents a currently hidden image.
  • .caption - The container for the caption.

As for the options configured via the JavaScript:

  • prefix (string) - A prefix to be appended in front of the class names. It can only contain alphanumeric characters and the characters - and _
  • images (array) - This array should contain objects which contain attributes set on the images, e.g. src, alt, title
  • statusPosition (string) - This can either be 'hide', 'after', 'before', 'bottom' or 'top'
  • showTabs (boolean) - Self explanatory.
  • showCaptions (boolean) - Self explanatory.
  • baseStyle (boolean) - Whether this gallery should make use of the default styling as a base.
  • allowClick (boolean) - Whether the tabs should be clickable.
  • speed (integer) - How many milliseconds it should take to change images.
  • delay (integer) - How many milliseconds the gallery should wait until changing image.

For example:

<div id="demo-styled"></div>
/* as this is the unique gallery element itself,
the class .gallery has not been appended to the selector,
but if you want to you can */
#demo-styled
{
	background-color: #000;
	width: 480px;
	height: 300px;
}

#demo-styled .images
{
	border: 1px solid #99a488;
}

#demo-styled .tab
{
	margin-top: 5px;
	width: 16px;
	height: 4px;
}
#demo-styled .tab-on
{
	background-color: #aaec41;
}
#demo-styled .tab-off
{
	background-color: #99a488;
}

#demo-styled .caption
{
	font-size: 11px;
	font-family: georgia, 'times new roman', times, serif;
	height: 13px;
}
$(document).ready(function() {
	$("#demo-styled").glyr({
		images: [
			{ src: 'rose1.jpg', alt: 'Rosa bracteata' },
			{ src: 'rose2.jpg', alt: 'Bandel rose' }
		],
		allowClick: false,
		showCaption: false,
		speed: 1000
	});
});
